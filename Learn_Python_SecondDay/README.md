\r \n：   表示回车和换行符\
\t：      制表符\
\\：      代表'\'本身\
\^：      匹配^符号本身\
\$：      匹配$符号本身\
\.：      匹配小数点（.）本身\
\d:       任意一个数字，0~9中的任意一个数字\
\w:       任意一个字母或者数字或下划线，也就是A~Z,a~z,0~9,_中的任意一个\
\s:       包括空格，制表符，换页符等空白字符的其中任意一个\
.:        小数点可以匹配除了换行符（\n）以外的任意一个字符\
